# DSA4262 AY25/26 Sem 2

## Overview
This repository contains Individual Assignments for DSA4262.

## Assignment 1 – Mental Health in Singapore

### Folder Structure
- `Assignment 1/`
  - `DSA4262_Assignment1.ipynb` – Main analysis and visualisations
  - `data/` – Raw and cleaned datasets used in the analysis

### Data Sources
- Ministry of Health (MOH), National Population Health Survey (NPHS)
- National Youth Mental Health Study (NYMHS)

### Methods
- Data cleaning and aggregation using pandas
- Visualisation using matplotlib and seaborn
- Confidence intervals included where available to reflect sampling uncertainty

### Reproducibility
To reproduce the analysis:
1. Open the Jupyter notebook in `Assignment 1/`
2. Run cells sequentially from top to bottom
